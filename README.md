# 麦块 for 我的世界 iOS 2.0

* 产品简介
麦块 for 我的世界，是Tars团队精心打造的专注于游戏Minecraft的社区论坛，收集了大量MC资源、资讯、攻略。加入进来，能让你和更多的MC玩家交流游戏乐趣和经验心得。

* 特色功能：
1.麦块社区完整的移动体验。
2.每日热门新闻和帖子精心推荐。
3.麦友聊天、攻略收藏尽在手中。

如果你对麦块有兴趣的话,欢迎fork并帮助我们改进.
